API BERITA

FITUR
1. Register dan Login
2. Menambah berita baru
3. Mengedit berita
4. Menghapus berita
5. Mengambil data berita terbaru dan berdasarkan id berita
6. Mencari berita berdasarkan kata kunci judul dan isi berita

TEKNOLOGI
1. Laravel
2. Sanctum
3. MySQL

INSTALLATION
1. Clone github
2. Composer install
3. Setup env database
4. Migrate database
5. php artisan serve
